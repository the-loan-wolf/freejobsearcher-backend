## Getting Started

### Prerequisites
Ensure you have the following installed:
- [Node.js](https://nodejs.org/en/download)
- [NPM](https://nodejs.org/en/download)

### Installation
1. **Clone the repository:**
   ```bash
   $ git clone https://github.com/the-loan-wolf/freejobsearcher-backend.git
   $ cd freejobsearcher-backend
   $ npm install
   $ npm run dev
   ```
